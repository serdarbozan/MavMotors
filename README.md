# MavMotors

A vehicle marketplace app built for UTA students. Only `@mavs.uta.edu` email addresses can register.

## Default Admin Account
- Email: admin@admin.com
- Password: admin

---

## Features

- Browse and search vehicle listings with filters (price, type, year, mileage)
- Save vehicles to a wishlist
- Add vehicles to cart and purchase via full payment or financing
- View purchased vehicles under **My Garage**
- Post and manage your own listings
- Dark / Light mode toggle in settings

## Admin Features

- View and manage all listings
- View, suspend, and delete users
- Audit log for all admin actions

## Tech Stack

- Kotlin
- Android Room (local database)
- Glide (image loading)
- Material Design components

## Requirements

- Android Studio
- Min SDK: API 24
- UTA student email (@mavs.uta.edu) required for registration
